# ⚽ 하프타임의 기적 — 0-2를 뒤집어라

월드컵 16강 하프타임 라커룸. 스코어 0-2. 당신에게 3분이 주어집니다.

지시 카드를 골라 전술을 결단하고, 후반전 하이라이트를 지켜보고, 75분 터치라인에서 마지막 지시를 내리세요. 같은 카드라도 75분의 결단이 결말(역전승 / 연장행 / 석패)을 가릅니다.

**▶ 플레이: https://hooniesl.github.io/halftime-miracle/**
(회원가입·설치·API 키 없이 브라우저에서 바로 실행됩니다. 모바일·데스크톱 모두 지원)

## 게임 흐름
1. **인트로** — 전반전 0-2 하이라이트 요약과 상대 스카우팅 노트
2. **라커룸** — 3분 제한시간 안에 지시 카드 12장 중 3~4장 선택. 카드를 고를 때마다 포메이션 보드의 선수 11명이 실시간으로 움직이고 공격·수비·체력 게이지와 시너지⚡/충돌💥가 즉시 반응
3. **후반전 재생** — 하이라이트 장면이 미니 피치 애니메이션+실황 중계로 흐르다가, **75분에 경기가 멈추고 긴급 지시(총공세/유지/잠그기)를 결단**
4. **감독 리포트** — 최종 결과·감독 평점·시너지 분석, "다른 선택이었다면?" 재도전, 내 경기 리플레이 링크 공유

## 기술
- **React 18 + Vite** — 순수 클라이언트 SPA (서버·로그인·외부 API 없음)
- **결정론 규칙 엔진** (`src/engine.js`) — 같은 카드 조합+개입 선택 = 항상 같은 후반전 (시드 고정 난수). 카드 상성·시너지가 결과를 만들고, 각 장면 중계 문구에 선택한 카드가 등장해 "왜 이 결과인지"가 보이게 설계
- 선수·팀 데이터는 자체 제작 가상 더미 JSON (`src/data.js`) — 실존 선수·국가 미사용
- 밸런스 자체 실측: 전 조합 715개 기준 역전승 24% / 연장행 22% / 석패 55%, 75분 개입으로 결말이 바뀌는 조합 35%

## 로컬 실행
```bash
npm install
npm run dev        # 개발 서버 (localhost:5173)
npm run build      # 프로덕션 빌드 (dist/)
node qa_smoke.mjs  # Playwright 브라우저 QA (vite preview 4197 필요)
```

